Welcome on Sdef Editor !

Scripting definition files (sdefs) are XML files that describe everything about an application's scriptability: terminology, implementation information, and complete documentation. Sdef Editor allows to simply create, and edit sdefs and it uses sdefs to generate a variety of implementation and definition files for use in an application and in AppleScript.

If you are familiar with AppleScript or with writing aete resources, most of the sdef elements will be familiar to you. If you are not, or if you want to know how sdef elements map to your implementation language, here is a brief translation guide. For more detailed information, see the definitions in the Reference section.

sdef code
class class
property attribute (XML), field (C), member variable (C++/ObjC), to-one relation
element element (XML), to-many relation
command, event verb, method (ObjC), member function (C++) (this is not entirely accurate; see the Reference entry)
parameter parameter
direct parameter self (ObjC) or this (C++), with some caveats; see the Reference entry
